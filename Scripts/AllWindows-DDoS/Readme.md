🆘 Troubleshooting
# Issue 1: Permission Denied
```
sudo chmod +x allwindows_ddos.sh
sudo ./allwindows_ddos.sh
```
# Issue 2: hping3 Not Found
```
sudo apt-get update
sudo apt-get install hping3 -y
```
# Issue 3: Target Not Reachable
```
# Check Windows Firewall - Turn it OFF
# Check VirtualBox Network - Set to Host-Only
# Test: ping 192.168.56.104
```
# Issue 4: Raw Socket Error
```
sudo setcap cap_net_raw+ep /usr/sbin/hping3
```
